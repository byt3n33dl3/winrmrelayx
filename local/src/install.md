## Installation

_Download:_
```sh
git clone https://github.com/byt3n33dl3/winrmrelayx.git
cd winrmrelayx
```

_Run Execution:_

```sh
python3 -m venv venv
pip3 -r install requirements.txt
```

This adversarial project are Impacket-based, make sure you have that installed first.
